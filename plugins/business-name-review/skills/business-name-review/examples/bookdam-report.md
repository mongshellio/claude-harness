## 4. KIPRIS 상표 출원

| 검색 키워드 | 결과 |
|---|---|
| `"북담" KIPRIS OR 상표 OR 출원` | 일반 정보 외 출원 이력 검색 결과 없음 |
| `"북담" 상표권 등록` | 동일 |
| `"bookdam" trademark` | 글로벌 결과 없음 |

**사용자 직접 확인 필요** (이게 결정적):
- https://www.kipris.or.kr 접속 → 상표 검색
- 검색어:
  - `북담` (한글 정확 일치)
  - `Bookdam` (영문 정확 일치)
- 확인할 클래스 (SaaS·책 카테고리):
  - **9류** (소프트웨어, 다운로드 앱) ⭐
  - **35류** (광고/사업관리/온라인소매) ⭐
  - **41류** (교육/오락/콘텐츠 제공) ⭐
  - **42류** (SaaS 호스팅) ⭐
  - 16류 (인쇄물·서적) — 책 카테고리 인접

**필요시**: 변리사 무료 상담 (특허청 IP-DESK)

**역사적 참조**: 류성룡의 시에 "북담정사(北潭精舍)"라는 학자의 서재 명칭이 등장. 일반명사 아닌 고유 의미로 한국 학문 전통에 닿아있음 → 브랜드 스토리 자산이지 충돌 아님.

## 종합 의견

**북담은 SaaS·AI·앱 카테고리에서 사실상 비어있는 이름**입니다. 글로벌·국내 직접 경쟁자가 없고, 도메인 검색 인덱싱도 모두 0건. 한국어 의미("책 + 담화")가 직관적이고 발음 쉬워서 한국 시장 우위.

**유일한 명확한 리스크**는 인스타그램 `@bookdam_bookdam` 계정. 같은 컨셉(책 관련)으로 이미 존재하므로 핸들 변형(예: `@bookdam.app`)으로 우회하거나, 활동성이 낮으면 사용자가 직접 컨택하여 협력·인수 가능. KIPRIS·상호 충돌은 본인이 5분 안에 직접 확인 가능한 수준.

**우선 액션 순위**:
1. **(오늘)** 후이즈에서 7개 TLD 가용성 확인 → 가용한 것들 즉시 등록 (`.com`+`.app`+`.ai`+`.co.kr` 최소)
2. **(오늘)** KIPRIS에서 "북담"/"Bookdam" 9·35·41·42류 직접 조회
3. **(이번 주)** SNS 핸들 후보 확정 → 전 플랫폼 동시 가입으로 선점
4. **(법인 설립 시점에)** 대법원 인터넷등기소 동일 관할 동일 분야 등기 확인
