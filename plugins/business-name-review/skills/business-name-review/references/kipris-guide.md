# KIPRIS 상표 검색 가이드

## KIPRIS란

특허청이 운영하는 한국지식재산정보원(KIPI)의 무료 검색 서비스. 한국에서 출원·등록된 모든 상표를 누구나 조회 가능.

- URL: https://www.kipris.or.kr
- 회원가입 불필요 (조회만 할 경우)

## 검색 절차

1. 메인 페이지 상단 → "상표" 탭
2. 빠른검색 또는 항목별검색
3. **검색어**: 후보 이름의 한글·영문·발음 변형 모두
   - 예: "북담", "Bookdam", "북담앱" 등
4. **검색결과 화면**에서 다음 항목 확인:
   - 출원/등록번호
   - 상태: 출원·등록·소멸·거절 등
   - 권리자(출원인)
   - 지정상품·지정서비스업
   - 상품분류(니스 분류)
   - 출원일/등록일/존속기간

## 상품·서비스 분류 (니스 분류)

상표권은 **클래스(분류) 단위로 권리 발생**합니다. 같은 이름이라도 클래스가 다르면 충돌 안 함.

SaaS/IT 프로덕트 관련 주요 클래스:

| 클래스 | 분야 | 우리 컨셉(이름 검토 SaaS) 관련성 |
|---|---|---|
| **9류** | 소프트웨어, 다운로드 앱, 전자출판물 | ⭐⭐⭐ (앱·소프트웨어 본체) |
| **35류** | 광고·사업관리·온라인소매 | ⭐⭐⭐ (SaaS 비즈니스) |
| **38류** | 통신서비스 (메신저/채팅 포함) | ⭐⭐ (대화형 서비스라면) |
| **41류** | 교육·오락·문화 (전자출판물 콘텐츠 제공) | ⭐⭐⭐ (책·교육 콘텐츠) |
| **42류** | 컴퓨터·소프트웨어 디자인·SaaS 호스팅 | ⭐⭐⭐ (SaaS 인프라) |
| **16류** | 인쇄물·서적 (종이책) | ⭐ (도서 콘텐츠) |
| **45류** | 법률/안전 서비스 | (관련 적음) |

**책 관련 챗봇/AI 앱**이라면 보통 **9, 35, 41, 42류**가 핵심.

## "충돌"의 의미

같은 이름이 같은(또는 유사한) 클래스에서 이미 출원·등록되어 있으면:
- **이미 등록(등록상표)**: 거의 확정적 거절. 출원해도 등록 불가. 사용 자체도 침해 위험.
- **출원 중(공고 전·심사 중)**: 등록될 수 있음. 등록되면 위와 같음.
- **소멸·거절·취하**: 권리 없어진 상태. 다시 출원 가능.
- **유사명·인접 클래스**: 심사관 판단에 따라 거절될 수 있음 — 변리사 상담 필요.

## 한국 외 추가 조회

- **미국**: USPTO TESS (https://www.uspto.gov/trademarks)
- **국제출원**: WIPO Madrid (https://www.wipo.int/madrid/monitor/en/)
- 한국 시장만 노린다면 KIPRIS만으로 충분, 미국·글로벌 진출 계획 있으면 USPTO·WIPO도 함께.

## 위험 시그널 판단법

- 🔴 **동일 이름 + 동일 클래스 + 등록 상태**: 사용 보류, 다른 이름 검토 권장
- 🟡 **유사 이름 + 같은 클래스** 또는 **동일 이름 + 인접 클래스**: 변리사 상담 후 결정
- 🟡 **소멸된 상표 동일 이름**: 사용 가능하지만 과거 권리자가 재출원할 가능성 (3년 이내 소멸이면 신중)
- 🟢 **검색 결과 없음**: 깨끗 — 다만 결과 0건이라고 자동 안전은 아니므로 인접 검색어로 한 번 더 확인

## 변리사 상담 옵션

- **무료 IP-DESK** (특허청 운영): 전국 21개 지역
- **한국지식재산보호원** 무료 자문: https://www.kipa.org
- **사설 변리사**: 통상 출원 1건당 30~50만원 수준 (자기 출원 + 공식 등록비)
