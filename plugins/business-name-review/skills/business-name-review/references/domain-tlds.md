# 도메인 TLD 우선순위 가이드

## 핵심 TLD 7종

이 skill은 기본적으로 다음 7개를 검토합니다.

| TLD | 분류 | 우선순위 | 비고 |
|---|---|---|---|
| `.com` | 글로벌 일반 | ⭐⭐⭐⭐⭐ | 사실상 표준. 가능하면 무조건 잡기 |
| `.co` | 글로벌 일반 (콜롬비아 국가도메인) | ⭐⭐⭐⭐ | `.com` 못 잡았을 때 차선. 짧고 깔끔 |
| `.app` | Google이 운영하는 앱 전용 | ⭐⭐⭐⭐ | HTTPS 강제 (HSTS preload). 앱·웹앱에 좋음 |
| `.io` | IT 스타트업 전통 | ⭐⭐⭐⭐ | 영국령 인도양 국가도메인이지만 IT에서 사실상 글로벌 |
| `.ai` | AI 제품 | ⭐⭐⭐⭐⭐ | AI 프로덕트면 거의 필수. 2024 이후 1년 등록 가능해지면서 가격 인하 (연 ~$60-100) |
| `.co.kr` | 한국 기업 | ⭐⭐⭐⭐ | 한국 시장 진출 시 필수. 사업자등록증 필요할 수 있음 |
| `.kr` | 한국 일반 | ⭐⭐⭐ | `.co.kr`보다 짧지만 인지도 약간 낮음 |

## 추가로 고려할 만한 TLD

| TLD | 케이스 |
|---|---|
| `.dev` | 개발자 도구·라이브러리. HTTPS 강제 |
| `.so` | 짧은 도메인 원할 때 |
| `.xyz` | 가격 저렴, 디파이/Web3 트렌드 |
| `.studio` | 스튜디오·에이전시 |
| `.book` / `.club` / `.shop` 등 신규 gTLD | 분야 특화. 가격 천차만별 |

## `.co.kr` vs `.kr` 차이

- `.co.kr` — **상업·기업용**. 보통 사업자등록증/법인 정보 등록. 한국 기업 신뢰도 ↑
- `.kr` — **일반용**. 개인도 등록 가능. 짧음. `.co.kr`보다 인지도 살짝 낮음
- 두 개 모두 잡아두는 것이 일반적 (방어 등록)

## 등록 비용 가늠

| TLD | 연간 비용(대략) |
|---|---|
| `.com`, `.net`, `.org` | $10-15 |
| `.co`, `.io` | $25-35 |
| `.app`, `.dev` | $15-20 |
| `.ai` | $60-100 |
| `.co.kr`, `.kr` | 연 22,000원~ (후이즈/가비아 기준) |

## 한국 등록업체

- **후이즈**: https://domain.whois.co.kr — 한국 점유율 1위, UI 한국어
- **가비아**: https://domain.gabia.com — 코스닥 상장, 안정성
- **Namecheap**: https://www.namecheap.com — 해외, 글로벌 TLD 저렴

`.com/.io/.app/.ai` 등 글로벌 TLD는 Namecheap이 가장 저렴, `.co.kr/.kr`은 후이즈·가비아.

## 도메인 잡을 때 체크리스트

- [ ] 우선순위 1순위 TLD (보통 `.com`) 가용성 확인
- [ ] 같은 이름으로 `.app`/`.ai`/`.io` 중 하나 추가 (앱·AI 제품이면)
- [ ] 한국 시장 진출 계획 있으면 `.co.kr` (+`.kr`) 동시 잡기
- [ ] **변형/오타 도메인** 방어 등록 (예: bookdam.app + bookdarm.app + book-dam.app)
- [ ] WHOIS 개인정보 보호 옵션 활성화 (가격 거의 없음)
- [ ] 1년 등록 ≠ 영구. 갱신 자동결제 또는 만료 알림 설정

## 우회 패턴 (1순위 TLD 못 잡았을 때)

원하는 도메인이 점유돼 있을 때 고려:

1. **다른 TLD**: bookdam.com 막혔으면 `bookdam.app`, `bookdam.ai`, `getbookdam.com`
2. **접두/접미**: `try-`, `get-`, `use-`, `-app`, `-hq`, `-co` 등
3. **하이픈**: `book-dam.com` (그러나 일반적으로 권장 안 함, 입력 번거로움)
4. **변형 철자**: `bookdam` → `bookdarm`, `bookdom` (브랜드 일관성 해침, 신중)
5. **인수 시도**: WHOIS로 소유자 정보 보고 직접 컨택 (가격 천차만별)
