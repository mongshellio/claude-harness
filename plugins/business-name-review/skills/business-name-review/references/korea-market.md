# 한국 시장 특화 검토 가이드

이름을 한국 시장에서 쓸 거면 글로벌 검토(도메인·인스타·X·KIPRIS)만으로는 부족하다. 한국 사용자들이 실제로 검색하고 발견하는 채널이 따로 있고, 사업자등록·법인 설립 시 별도 충돌 위험이 있다.

## 1. 발견(Discovery) 채널 — SNS 외 한국 핵심 채널

### 네이버 카페

- **검색 방법**: `site:cafe.naver.com "<name>"`
- **왜 중요**: 한국에서 brand discovery의 30%+를 차지. 동명 카페가 활성화돼 있으면 신규 브랜드가 검색 결과에서 밀림
- **위험 시그널**:
  - 활성 카페(회원 1만+) + 같은 카테고리 → 🔴
  - 활성 카페 + 다른 카테고리 → 🟡 (SEO 압력)
  - 비활성·소규모 카페 → 🟢

### 네이버 블로그

- **검색 방법**: `site:blog.naver.com "<name>"`
- **왜 중요**: 한국 검색 1위 네이버에서 블로그가 상위 노출 비중 큼
- **체크**: 동일 이름의 파워블로그가 있으면 SEO 경쟁 발생

### 카카오톡 채널

- **검색 방법**: 
  - WebSearch: `site:pf.kakao.com "<name>"`
  - 또는 사용자가 카카오톡 앱에서 "<name>" 검색
- **왜 중요**: 한국 고객 acquisition·CS 핵심 채널. 동명 채널이 있으면 채널명 변형 필요
- **참고**: 카카오톡 비즈니스 채널명은 사업자등록 필요

### 인스타그램·유튜브 (한국 사용자 채널 포함)

글로벌 검색과 별개로 한국어 콘텐츠도 따로 봐야 함.
- 인스타: `site:instagram.com "<handle>"` 결과에서 한국어 설명·게시물이 있는 계정 별도 표시
- 유튜브: 한국 채널은 채널명이 한글일 때가 많음. `site:youtube.com "<한글 이름>"`도 별도 검색

## 2. 사업자·법인 상호 중복 검토

### 개인사업자 상호

- **bizno.net** — 사업자등록번호·상호 통합 검색 (https://www.bizno.net)
  - 검색: `"<name>" site:bizno.net`
  - 또는 사이트 직접 접속 후 상호 검색
- 같은 시·도 내 동일 상호 사업자가 있어도 **법적으로 등록 가능** (개인사업자는 상호 중복 제한 거의 없음)
- 다만 **세무·결제·검색 혼동** 위험 → 변경 권장

### 법인 상호 등기

- **대법원 인터넷등기소** (https://www.iros.go.kr)
- 법인 설립 시 같은 관할 등기소에 **동일 상호 + 동일 영업 분야**가 등기돼 있으면 **상법 제22조**에 의해 등기 거부 가능
- 자동 검색 어려움 → 사용자에게 직접 조회 안내

### 국세청 사업자등록상태 조회

- https://teht.hometax.go.kr → 사업자등록상태 조회
- 본인 직접 조회용 (특정 사업자등록번호 기반)
- 신규 등록 단계에선 적용 X. 이미 등록된 사업자 상태 체크 용도.

## 3. 한국 시장 검색 평가 체크리스트

다음 항목을 모두 봐야 한국 시장 검토가 완성:

- [ ] 네이버 카페 동명·동분야 검색
- [ ] 네이버 블로그 동명 인기 블로그 검색
- [ ] 카카오톡 채널 동명 검색 (사용자 직접 확인 요청)
- [ ] 유튜브 한국 채널 동명 검색
- [ ] bizno.net 또는 국세청에서 동일 상호 사업자 존재 여부
- [ ] (법인 설립 계획 시) 대법원 등기소에서 동일 상호 동일 분야 검색

## 4. 한국 시장 특화 위험 패턴

### 외국어 발음 충돌
- 영문 이름이 한국어로 다른 의미가 되는 경우 (예: "Pia" → 한국어 발음 부정적 연상)
- 한글 이름이 영어권에서 부정적 연상되는 경우

### 동음이의어
- "북담"의 경우 "북담정사" 같은 역사적 의미가 있음 — 일반적으론 자산이지만 종교·문화 단체와 헷갈릴 수 있음 확인

### 카테고리 인식 차이
- 한국에서 "북" = 책 vs 드럼 vs 북쪽 → 카테고리 명확성 약화. 본인 이름 조합이 이런 모호성을 만드는지 체크
